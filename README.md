> Uma Landing Page de alta conversão e imersão, projetada especificamente para uma instituição de ensino especializada no acolhimento de pessoas neurodivergentes (TDAH e Autismo) da educação infantil à faculdade.

## 🚀 Sobre o Projeto

O **Instituto Aquarela** não é apenas um site escolar genérico. Ele é o resultado de um estudo focado em produto e copywriting para atingir diretamente as dores reais de famílias de crianças neurodivergentes e jovens adultos que lidam com disfunção executiva. 

A interface foi projetada com extremo cuidado focado na **acessibilidade (A11y)**, utilizando de uma paleta de cores acolhedora para evitar superestimulação visual e um design premium baseado no conceito de *Glassmorphism*.

## ✨ Principais Funcionalidades

* **Painel de Acessibilidade (A11y):** Sistema customizado com persistência de dados via `localStorage` que permite ao usuário aumentar a fonte da página ou ativar um Modo de Alto Contraste instantâneo.
* **Menu Hamburguer e Glassmorphism:** Barra de navegação totalmente responsiva com painel lateral oculto (Mobile Drawer) e detalhes em vidro translúcido nas interfaces.
* **Animações de Scroll (Fade In):** Animações criadas via JavaScript puro (`Intersection Observer API`) que suavizam a entrada de elementos. Efeito de *Parallax* implementado na seção inicial.
* **Equipe Especialista (Modais):** Seção de prova social contendo interações que acionam janelas flutuantes (Modais) de forma limpa.
* **Carrossel de Depoimentos Automático:** Integração fluída com a biblioteca **Swiper.js** para slider *touch-friendly*.
* **Contato Inteligente (Formspree):** Formulário completo e funcional, estruturado para capturar leads de forma autônoma sem necessidade de um backend complexo, direcionando os e-mails para a caixa da instituição.
* **FAQ Sanfona (Accordion Nativo):** Perguntas frequentes construídas com extrema leveza utilizando as tags semânticas `<details>` e `<summary>`.
* **SEO & Open Graph:** Código devidamente tagueado no `<head>` para garantir um compartilhamento perfeito com imagens miniatura (thumbnails) no LinkedIn e WhatsApp.

## 🛠️ Tecnologias Utilizadas

* **HTML5 Semântico**
* **CSS3** Avançado (Variáveis de CSS, Grid, Flexbox, Animations)
* **JavaScript (Vanilla)**
* **Swiper.js** (CDN)
* **Formspree** (API)
* Imagens hiper-realistas geradas por Inteligência Artificial

## 💻 Como Rodar o Projeto

Como o projeto foi desenvolvido focando em performance bruta (Vanilla), a visualização é instantânea e não requer instalação de pacotes (`npm`).

1. Faça o clone do repositório:
```bash
git clone https://github.com/jonatascavalcanti03/instituto-aquarela.git
```
2. Entre na pasta do projeto.
3. Abra o arquivo `index.html` em qualquer navegador web moderno.
